CSS (cascading style sheets) is a language used to customize a website. You can implement it to change the color,font,size,etc of text, background,and more in a website/webpage to make it what you want it to be.Internal css goes with the head of the html document and you use the style tag to write the css. External css uses a css file. In that file you dont need to use angle brackets and you can group the things that you want to customize together if they are going to use the same thing. Inline css you go to the opening tag of the HTML code and put the css code and attributes in there to change the background color, text.etc. All 3 types of css uses {}, background-color, font-family,and color.
